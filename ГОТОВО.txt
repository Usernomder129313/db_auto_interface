ВАРИАНТ 1: Из архива (РЕКОМЕНДУЕТСЯ)
───────────────────────────────────────────────────────────────────────────────
   1. Скопируйте файл: dist/db-auto-interface-1.0.0.tar.gz
   2. На новом устройстве выполните:
      pip install db-auto-interface-1.0.0.tar.gz

ВАРИАНТ 2: Из wheel пакета
───────────────────────────────────────────────────────────────────────────────
   1. Скопируйте файл: dist/db_auto_interface-1.0.0-py3-none-any.whl
   2. На новом устройстве выполните:
      pip install db_auto_interface-1.0.0-py3-none-any.whl

ВАРИАНТ 3: Из исходников
───────────────────────────────────────────────────────────────────────────────
   1. Скопируйте всю папку: db_auto_interface_package
   2. На новом устройстве выполните:
      cd db_auto_interface_package
      pip install .

После установки проверьте:

1. Импорт:
   python -c "from db_auto_interface import DBAutoInterface; print('OK')"

2. Версия:
   python -c "from db_auto_interface import __version__; print(__version__)"

3. Установка:
   pip list | grep db-auto-interface

4. Консольная команда (если установлена):
   db-auto-interface --help
