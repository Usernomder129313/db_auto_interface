🚀 КАК УСТАНОВИТЬ НА ДРУГОМ УСТРОЙСТВЕ
═══════════════════════════════════════════════════════════════════════════════

ШАГ 1: Скопируйте архив
───────────────────────────────────────────────────────────────────────────────
   Скопируйте файл: dist/db-auto-interface-1.0.0.tar.gz
   
   Или wheel: dist/db_auto_interface-1.0.0-py3-none-any.whl

ШАГ 2: Установите на новом устройстве
───────────────────────────────────────────────────────────────────────────────
   pip install db-auto-interface-1.0.0.tar.gz
   
   Или:
   pip install db_auto_interface-1.0.0-py3-none-any.whl

ШАГ 3: Проверьте установку
───────────────────────────────────────────────────────────────────────────────
   python -c "from db_auto_interface import DBAutoInterface; print('OK')"

ШАГ 4: Используйте в коде
───────────────────────────────────────────────────────────────────────────────
   from db_auto_interface import DBAutoInterface
   
   app = DBAutoInterface(connection_params={
       'host': 'localhost',
       'database': 'your_db',
       'user': 'postgres',
       'password': 'password',
       'port': 5432
   })
   app.run()
